# AI-Based Product Return Prediction System

## 📌 Problem

E-commerce platforms face high return rates due to:

* Size mismatch
* Misleading product descriptions
* Poor recommendations

## 🎯 Objective

Build an AI system to predict return probability before purchase.

## 🚀 Solution

This project uses a Machine Learning model to:

* Analyze product & user factors
* Predict likelihood of return
* Help users make better decisions

## 🧠 Features

* Return prediction using Logistic Regression
* Simple input-based interface
* Scalable for real-time recommendation systems

## 📊 Input Features

* Size match (0/1)
* Description match (0/1)
* Product price

## 📦 Output

* Return probability (High / Low)

## ⚙️ Installation

```bash
git clone https://github.com/your-username/return-prediction-ai.git
cd return-prediction-ai
pip install -r requirements.txt
```

```bash
python app.py
```
