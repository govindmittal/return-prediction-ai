# Installation Guide

## 📦 Prerequisites

* Python 3.8 or above
* pip (Python package manager)

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/govindmittal/return-prediction-ai.git
cd return-prediction-ai
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
python app.py
```

## ✅ Expected Output

* The system will ask for inputs like size match, description match, and price
* It will predict whether the product is likely to be returned

## 🛠 Troubleshooting

* If pip is not working, try:

```bash
python -m pip install -r requirements.txt
```
